# 🚀 eventamikom3368 - EventHub3368

Selamat datang di repository **EventHub3368**. Project ini adalah aplikasi manajemen event yang dibangun menggunakan framework **Laravel** untuk kebutuhan perkuliahan di **Amikom Yogyakarta**.

---

## 📄 Deskripsi Project
Project ini bertujuan untuk mengelola data event secara digital dengan sistem yang terintegrasi. Fokus utama pengembangan adalah pada kemudahan input data dan keamanan informasi mahasiswa.

## 🛠️ Stack Teknologi
* **Framework:** Laravel 11
* **Language:** PHP
* **Package Name:** `marom/eventhub3368`
* **OS:** macOS (MacBook Pro)

## 📦 Cara Menjalankan Project (Local)
Jika kamu ingin mencoba project ini di perangkat lain, ikuti langkah berikut:

1. **Clone Repository:**
   `git clone https://github.com/lulunovii/eventamikom-3330.git`
2. **Install Dependencies:**
   `composer install`
3. **Setup Environment:**
   `cp .env.example .env` && `php artisan key:generate`
4. **Run Server:**
   `php artisan serve`
